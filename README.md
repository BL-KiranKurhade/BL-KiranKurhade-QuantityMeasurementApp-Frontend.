# UC20 - React Frontend for Quantity Measurement App

## Prerequisites
- Node.js 16+
- npm 8+

## Install & Run
```bash
npm install
npm start
```
Opens at http://localhost:3000

## Features
- Unit Converter (Length, Weight, Volume, Temperature)
- Live API integration with UC17 backend
- JWT Authentication + Google OAuth2 (via UC18)
- Material UI components
- React Router navigation
- Custom useConverter hook
- Session management with sessionStorage

## Build for Production
```bash
npm run build
```
